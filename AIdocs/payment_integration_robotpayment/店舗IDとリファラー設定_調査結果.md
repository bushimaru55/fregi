# 店舗ID（aid）・リファラー設定 調査結果

公式ドキュメントおよびウェブ上の情報を基に、**店舗IDの確認場所**と**リファラー（決済データ送信元URL）の設定場所**を整理しました。

---

## 結論（要点）

1. **請求管理ロボ（demo.billing-robo.jp）** と **決済システムコントロールパネル（credit.j-payment.co.jp）** は**別システム**です。
2. **店舗ID（aid）** は **決済システムコントロールパネル** で確認します。請求管理ロボの「API接続詳細」には**表示されません**。
3. **リファラーエラー**を解消する「決済データ送信元URL」は、**決済システムコントロールパネル**の「設定」→「決済システム」→「PC用決済フォーム設定」で設定します。

---

## 1. 公式出典

### 1.1 請求管理ロボ API 仕様書（クレジットカード登録・トークン方式 3Dセキュア）

- **URL**: https://apispec.billing-robo.jp/public/billing_payment_method/credit_card_token.html  
- **記載内容**:
  - トークン取得時の `aid` は「**店舗ID ※契約時に発行された決済システムの店舗ID**」で、6桁数値必須。
  - 設定が完了後、**コントロールパネル**で以下を設定するよう記載:
    1. 「**設定**」→「**決済システム**」
    2. 「決済ゲートウェイ＆CTI決済設定」→「決済データ送信元IP」
    3. 「**PC用決済フォーム設定**」→「**決済データ送信元URL**」
  - ここでいう「コントロールパネル」は、店舗IDを扱う**決済システム**側のCPを指すと解釈可能。

### 1.2 MyASP マニュアル（ROBOT PAYMENT決済）

- **URL**: https://docs.myasp.jp/19489/  
- **記載内容**:
  - 「**ROBOT PAYMENTコントロールパネル側に記載されている「店舗ID」を入力**します。」
  - 「[ROBOT PAYMENTコントロールパネル](https://credit.j-payment.co.jp/cp/signin.aspx) にログインします。**「設定」⇒「決済システム」**ページに移動し、「決済連携設定」画面に表示されている内容を設定してください。」
  - よって、**店舗IDの確認**と**決済システム関連の設定**は **credit.j-payment.co.jp** のコントロールパネルで行うことが明記されている。

### 1.3 決済システムコントロールパネルのURL

- **ログインURL**: https://credit.j-payment.co.jp/cp/SignIn.aspx  
- ROBOT PAYMENT 社の決済ゲートウェイ用管理画面。請求管理ロボ（billing-robo.jp）とは別ドメイン・別システム。

## 2. 請求管理ロボと決済システムCPの役割整理

| 項目 | 請求管理ロボ（billing-robo.jp / demo.billing-robo.jp） | 決済システムCP（credit.j-payment.co.jp） |
|------|--------------------------------------------------------|------------------------------------------|
| 主な用途 | API（user_id, access_key）、請求先・決済情報、通知URL、Webhook など | 店舗ID（aid）、決済データ送信元IP/URL（リファラー） |
| 店舗ID | **表示されない** | **ここで確認する** |
| 決済データ送信元URL | 設定なし | 「設定」→「決済システム」→「PC用決済フォーム設定」→「決済データ送信元URL」 |
| 決済結果通知URL | ここで登録 | （通知は請求管理ロボ側で受ける想定） |

---

## 3. デモ環境で credit.j-payment.co.jp にログインできない場合

- デモ契約によっては、**請求管理ロボ**のみ提供され、**決済システムコントロールパネル**へのログイン権限が無い場合があります。
- その場合は、以下を **ROBOT PAYMENT カスタマーサポート** に依頼してください:
  - **店舗ID（aid）** の確認・案内
  - **ローカル開発用の決済データ送信元URL**（例: `http://localhost:8080`）の登録
- **連絡先**: TEL 03-4405-0665（平日9:00–18:00）、Mail support@billing-robo.jp  
  （上記は請求管理ロボ API 仕様の「クレジットカード登録(トークン方式 3Dセキュア利用)」に記載のカスタマーサポート情報）
