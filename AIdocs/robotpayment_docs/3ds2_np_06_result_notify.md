# 決済結果通知のパラメータ仕様

## キックバック（rt=0）
- 形式：URLパラメータ
- 通知先：管理画面「決済結果通知設定」の「決済結果通知URL」
- 通知の成功判定を行っているため、通知先は **ContentLength が 0 以上** を出力するよう実装します。

### サンプル
```
決済結果通知URL?gid=10000001&rst=1&ap=0001112&ec=&god=10000002&cod=test&am=1000&tx=80&sf=500&ta=1580
```

## レスポンス（rt=1）
- 形式：HTML出力（カンマ区切り）

### サンプル
```
1095729,1,TestMod,,205685,,1000,2,4,1006,,,
0,2,,ER900,,,0,0,0,0,,,0,
```

## レスポンス項目（出力順）
|項目|フィールド|詳細|形式|出力順|
|---|---|---|---|---:|
|決済番号|gid|決済ごとに発行される番号|半角数字|1|
|決済結果|rst|1:決済成功 / 2:決済失敗|半角数字（1）|2|
|カード会社承認番号|ap|カード会社から発行される承認番号|半角英数（6～7）|3|
|エラーコード|ec|エラーコード|半角英数（12）|4|
|オーダーコード|god|決済ごとに発行されるコード|半角数字|5|
|店舗オーダー番号|cod|決済リクエスト時に指定したcod|50バイト以内|6|
|決済金額|am|決済リクエスト時に指定したam|半角数字|7|
|税金額|tx|決済リクエスト時に指定したtx|半角数字|8|
|送料|sf|決済リクエスト時に指定したsf|半角数字|9|
|合計金額|ta|am, tx, sf の合計値|半角数字|10|
|発行ID|id|弊社システムより発行されたID（※1）|半角英数（50字以内）|11|
|発行パスワード|ps|弊社システムより発行されたパスワード（※1）|半角英数（50字以内）|12|
|その他|（任意）|決済リクエスト時に指定したその他データ|1024バイト以内|13|

### 注記
- ※1 コンテンツ（ID/PW）を商品登録した商品で「商品登録あり決済」をした場合のみ返却。
