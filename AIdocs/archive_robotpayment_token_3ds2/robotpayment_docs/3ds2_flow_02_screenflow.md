## 5. 3Dセキュア2.0 画面フロー（ユーザー視点）

※画面名称は仕様図の意図を踏まえて文章化しています（表示内容はカード会社等により差異があります）。

1. **購入内容確認画面**  
   - ユーザーが購入ボタンを押下

2. **カード情報入力画面**  
   - ポップアップ方式：決済会社提供の入力画面がポップアップで表示  
   - カスタマイズ方式：加盟店サイト内の入力画面で入力

3. **3Dセキュア2.0 認証（リスクベース認証）**
   - **フリクションレス**：本人認証画面が表示されずに次へ進む  
   - **チャレンジ**：本人認証画面（ワンタイムパス等）が表示され、ユーザーが入力して認証

4. **決済完了画面**  
   - 認証・決済結果に応じて完了（もしくはエラー）表示
