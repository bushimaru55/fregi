## 重要ポイント（事故防止のための最小要約）
- **自動課金の初回決済**には **トークン作成 + 3DS2認証** が必須（P.3, P.6）。  
- 初回決済の接続先（`gateway_token.aspx`）は **加盟店サーバ（固定IP）からリクエスト**する必要あり。ブラウザから送ると **ER003（送信元IPエラー）**（P.6, P.9）。  
- 初回決済結果通知は **「決済結果通知URL」** へ通知される（**自動課金結果通知URLではない**）点に注意（P.10）。  
- 自動課金（2回目以降）の通知は、`rt` に関係なく **常にキックバック**（P.13）。
