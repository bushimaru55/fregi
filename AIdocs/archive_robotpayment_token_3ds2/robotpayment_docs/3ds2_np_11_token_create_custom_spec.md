# トークン作成（カスタマイズ方式）仕様

## 配布JS
- 接続先: https://credit.j-payment.co.jp/gateway/js/CPToken.js
- ファイル: CPToken.js

## 関数
- `CPToken.TokenCreate(tokenRequest, callback)`

### TokenCreate 引数仕様
|引数名|説明|備考|
|---|---|---|
|tokenRequest|トークン作成に必要なパラメータ|JSON形式|
|callback|実行後にコールバックされる関数|※`ThreeDSAdapter.authenticate` 関数を指定|

### tokenRequest 仕様
|項目|フィールド|詳細|必須|形式/制約|
|---|---|---|---|---|
|店舗ID|aid|契約時に発行される店舗のID|〇|半角数字（6）|
|カード番号|cn|カード番号|〇|半角数字|
|カード有効期限|ed|有効期限（YYMM）|〇|半角数字|
|カード名義（名）|fn|名義（名）|〇|半角英数記号（43）※1|
|カード名義（姓）|ln|名義（姓）|〇|半角英数記号（43）※1|
|セキュリティコード|cvv|VISA/MASTER/JCB/DINERS=3桁、AMEX=4桁|※2|半角数字|
|支払方法|md|未指定=一括。「10」一括、「61」分割、「80」リボ| |半角数字|
|分割回数|pt|「3,5,6,10,12,15,18,20,24」から選択|※4|半角数字|

#### 注記
- ※1 記号はピリオド/ハイフン（半角）。fn+ln の合計が44文字以内。
- ※2 セキュリティコード利用の場合は必須（契約が必要）。
- ※3 分割/リボは契約時のみ指定可。
- ※4 md=61（分割払い）指定時は必須。

## レスポンス
- なし（処理完了時に HTML の `id=tkn` 要素の `value` にトークン文字列が設定）

## その他
- 実装の際に `rpemvtds` クラスを使用しないでください（JS内で使用しているため）。
